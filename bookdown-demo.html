2 Introduction

R is an open-source statistical programming language. It is available from the following website: R-Cran. You should install R on you laptop. For us it is probably best to use the following link. You can also install R-Studio which is an Integrated Development Environment (IDE) for R.

2.1 Competitors

R is one of the main programmable statistical languages. Others include: * Stata: Use in academic econometrics. This is not open source. * Matlab: Used extensively in academic finance. It is not open-source. There is an open-source clone called Octave. It can be downloaded here. * Python: Computer Language for statistical programming.
* Julia: New language that is faster than R * C++: Programming language. Underneath R is C++ in some cases.

2.2 Advantages of using R

The main advantages of using an open-source statistical programming languages like R are: * It can be used by anyone for no cost. Your model will not depend on the next institution having the same software licence * Programming language will encourage reproducible research

Reproducible Research means that results can be easily replicated. This means * Mistakes are less likely to happen than would be the case with copy and paste * Mistakes can be more swiftly identified * Changes to data or research methods are easier to make

Some additional information on R, R-Studio and Reproducible Research * LSE commentary on Reinhart and Rogoff

Individual components of a vector, matrix or data.frame can be identifies by using square brackets and a pair of numbers with the first equal to the row and the second equal to column. You only need the first number for a vector.

A <- c(1, 2, 6, 8, 10)
B <- c(55, 56, 60, 20, 18)
C <- data.frame(A, B)
C[3, 2]
## [1] 60
C$A
## [1]  1  2  6  8 10

As you can see, a $ can also be used to identify a column by `name’.

3.4 Plotting

The plot function will allow you to create graphs with the data that you have.

x<-seq(-pi,pi,length=10)
y<-sin(x)
plot(x,y)
lines(x,y)

If you change the length of the x variable to 100 you will get a smoother line.

x<-seq(-pi,pi,length=100)
y<-sin(x)
plot(x,y)
lines(x,y)

z<-rnorm(100)
hist(z, prob=TRUE)
mu<-mean(z)
sig<-sd(z)
x<-seq(-4,4,length=500)
y<-dnorm(x,mu,sig)
lines(x,y, col='red')

3.5 Input data

By far the easiest way to input data is to read the data from a csv file. There are packages for reading from excel and from other databases that we will look at later.

da <- read.csv('./Data/bac.csv')
head(da)
##         Date   BAC    SPY
## 1 03/12/2012 11.58 139.59
## 2 01/11/2012  9.82 138.36
## 3 01/10/2012  9.28 137.58
## 4 04/09/2012  8.79 140.13
## 5 01/08/2012  7.95 136.66
## 6 02/07/2012  7.30 133.32
str(da)
## 'data.frame':    156 obs. of  3 variables:
##  $ Date: Factor w/ 156 levels "01/02/2000","01/02/2001",..: 152 91 81 156 65 119 47 37 112 20 ...
##  $ BAC : num  11.58 9.82 9.28 8.79 7.95 ...
##  $ SPY : num  140 138 138 140 137 ...

It is important to make sure that the variables in the data.frame of of the correct class. In this case, the Date series (first column) is a factor. We need to change it to date.

da$Date <- as.Date(da$Date, format = "%d/%m/%Y")
str(da)
## 'data.frame':    156 obs. of  3 variables:
##  $ Date: Date, format: "2012-12-03" "2012-11-01" ...
##  $ BAC : num  11.58 9.82 9.28 8.79 7.95 ...
##  $ SPY : num  140 138 138 140 137 ...
